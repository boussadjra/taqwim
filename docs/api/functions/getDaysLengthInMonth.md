# Function: getDaysLengthInMonth()

> **getDaysLengthInMonth**(`hy`, `hm`): `number`

Returns the length of a month in the Hijri calendar.

## Parameters

• **hy**: `number`

The Hijri year.

• **hm**: `number`

The Hijri month (1-12).

## Returns

`number`

The length of the month in days. Returns -1 if the month is invalid.
